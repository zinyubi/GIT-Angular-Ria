#LuciadRIA Geometry Package
This package contains the different constructs for constructive geometry operations.
Install this package using 

```
npm install @luciad/ria-geometry --save
```

##Documentation
For extra information, please consult the documentation shipped with your LuciadRIA release or check the <a href="dev.luciad.com">developer platform</a>.
